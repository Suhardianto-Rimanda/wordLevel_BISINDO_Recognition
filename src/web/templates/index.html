    Tentang Project

    Aplikasi web yang mengenali isyarat BISINDO dari webcam, menyusunnya menjadi kalimat, dan membacakannya sebagai suara. Tersedia pula panel suara → teks agar percakapan dengan pengguna tunarungu bisa berjalan dua arah. Dibangun sebagai Tugas Akhir D-III Teknik Informatika.

    Tech yang Digunakan

    • YOLOv8 sebagai Region of Interest (ROI)
    • MediaPipe Holistic ekstraksi landmark tangan
    • LSTM klasifikasi kata isyarat dari urutan gerakan
    • NLP rule-based perangkaian kata menjadi kalimat
    • gTTS sintesis suara dari kalimat
    • Web Speech API suara → teks (murni di browser)
    • Flask server + streaming MJPEG webcam

    Panduan pakai

    1. Buka Translator dari menu di atas.
    2. Mulai Kamera lalu izinkan akses kamera di browser.
    3. Peragakan isyarat di depan kamera dengan pencahayaan cukup.
    4. Baca hasil: kata terdeteksi (LSTM) & kalimat (NLP) di panel kanan.
    5. Dengarkan: aktifkan auto-play atau tekan “Putar Ulang”.
    6. Reset Model untuk mengosongkan kalimat dan mulai dari awal.

    Panel suara → teks

    1. Tekan Mulai Bicara dan izinkan mikrofon.
    2. Ucapan lawan bicara muncul sebagai gelembung chat.
    3. Bersihkan untuk mengosongkan riwayat percakapan.
